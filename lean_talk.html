How to enter common symbols

Both in VS Code and in Live Lean 4 online, you can enter commonly used symbols like this:
  • for ℕ ℤ ℚ ℝ ℂ, use \N \Z \Q \R \C
  • for ≤ ≥ use \le \ge
  • for ¬ ∨ ∧ → use \not \or \and \to
  • for ∀ ∃ use \ex \all
  • for x₂ use x\2
  • for α β γ π use \alpha \beta \gamma \pi
